Data sources and corrections

Provider information is checked, but it can change.

This is a soft-launch database. Please confirm provider details before relying on them, especially cost, eligibility, referral rules, and current availability.

Where provider information comes from

Source type How it is used
Provider-owned public pages Used for contact details, service focus, costs, and availability notes when published.
Healthpoint and trusted directories Used for provider-specific pages, service details, GP clinics, and local support routes.
Health NZ, ACC, WINZ, and public agencies Used for crisis guidance, funding information, public services, and eligibility pathways.
Professional directories and registers Used as supporting evidence only when public practice contact information is available.
Third-party public GP listings Used to improve GP practice coverage, with manual review and Healthpoint fallbacks where needed.

What the metadata means

  • Last verified means the month the public source was last checked or imported.
  • Confidence is an honest signal about the source, not a promise that the provider is available.
  • Needs manual verification means the listing should be checked by a person before stronger claims are made.

What to report

Please report anything that looks wrong or outdated, including:

  • phone, text, email, website, or address details
  • costs, funding options, or WINZ/ACC eligibility notes
  • whether a provider is taking new clients or referrals
  • service type, cultural support, language support, or age eligibility
  • providers that should not be listed publicly

Correction pathway

Send the provider name, what is wrong, the correct information if known, and the source URL or contact person that supports the correction.
