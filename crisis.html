Crisis and emergency guidance

If safety is uncertain, use urgent help now.

This page is general guidance. It is not emergency triage and does not replace 111, 1737, a crisis team, a GP, or an emergency department.

Call 111 now if there is immediate danger

Call 111 if someone might seriously hurt themselves or someone else, has already been harmed, is missing and at risk, has taken an overdose, is experiencing a medical emergency, or cannot stay safe.

Call or text 1737 if you need to talk now

Free call or text 1737 any time in New Zealand to talk with a trained counsellor. This can be useful if you can stay safe but need immediate support, help calming things down, or help deciding the next step.

Other urgent options

  • Go to the nearest emergency department if someone is medically unsafe.
  • Contact your local mental health crisis team if you already have their number.
  • Call your GP practice and ask for urgent mental health advice if it is open.
  • If you are worried about someone else, stay with them if it is safe and call 111 if risk increases.

Using this site during distress

If filling out the guide feels too much, stop and use 1737 or 111. The provider finder is for making a first contact, not for deciding whether a situation is safe.
